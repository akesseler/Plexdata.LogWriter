Stream Settings

The stream logger settings are the settings used together with each of the stream logger implementations. These settings are defined in interface IStreamLoggerSettings.

  • Logging Type
  • Stream
  • Encoding

Logging Type

The stream logger uses JSON message formatting as its default logging type. In other words, the constructor of the default implementation of interface IStreamLoggerSettings resets the logging type of its base class constructor.

Stream

The property Stream allows getting and setting the instance of a Stream derived class. Be aware, this property is indeed part of the setting interface, but it can never be initialized by an external configuration using ILoggerSettingsSection. Finally note, nothing happens as long as property Stream is set to null, neither an Exception nor writing of logging message.

Encoding

The property Encoding allows to change the file encoding to be used. The default value is UTF–8 but might be changes by derived classes.
