Console Settings

The console logger settings are the settings used together with each of the console logger implementations. These settings are defined in interface IConsoleLoggerSettings.

  • Use Colors
  • Window Title
  • Quick Edit
  • Dimension
  • Coloring

Use Colors

This property enables or disables the coloring of messages. Text coloring is a special feature of a console window and may help to distinguish different message type from each other. The default value is ON but might be changes by derived classes.

Window Title

This property gets and sets the title of the console window. Modifying the title of the console window might be useful to distinguish a console logging window from other console windows. The default value is empty but might be changes by derived classes.

Quick Edit

This property enables or disables the quick–edit mode of a console window. The quick–edit mode represents a special feature of the Windows console and allows to mark and copy text. The default value is OFF but might be changes by derived classes. Be aware, this feature is only available on Windows platforms!

Dimension

This property gets and sets the dimension of the console window. The dimension (number of rows and columns) represents another special feature of a console window and defines the size of underlying output buffer. The default value is Dimension(0,0) but might be changes by derived classes.

Coloring

This property allows changing the list of colors to be used for different message types. The coloring list is actually a dictionary with the logging level as access key. Each logging level is assigned to a particular color setup for the foreground and the background. The default coloring is shown below but might be changes by derived classes.
