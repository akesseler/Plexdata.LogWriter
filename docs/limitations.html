Limitations

At the moment there are some known limitations that will be pointed out in this chapter.

Console Logger for Windows

As already mentioned, the Console Logger for Windows cannot be used on platforms other than Windows. The reason behind, this logging writer uses functions of the Win32–API directly. These functions are not available on other platforms.

Persistent Logger Settings

It is no longer required that the path of a provided filename must exist. But a process must still have write access to the existing parent directory.
