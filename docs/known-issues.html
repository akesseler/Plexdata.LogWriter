Known Issues

At the moment there are some known issues that will be pointed out in this chapter.

Graylog Support

Please be aware, the Graylog support was implemented solely based on the official documentation of Graylog Extended Log Format (GELF) that is available on the Internet under https://go2docs.graylog.org/5-0/getting_in_log_data/gelf.html.

Conversely, this implementation has never been tested with a physical Graylog server! Unfortunately, this circumstance makes the implementation error-prone. Therefore, do not hesitate and immediately report any issue that occurs together with the Graylog/GELF implementation.
